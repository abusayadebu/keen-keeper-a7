# 👥 KeenKeeper — Keep Your Friendships Alive

KeenKeeper is a responsive friendship management web app that helps users track interactions with friends through calls, texts, and video chats. It also provides analytics and timeline history to maintain stronger relationships.

---

## 🛠️ Tech Stack

- React.js
- React Router DOM
- Tailwind CSS
- Context API
- Recharts
- JavaScript (ES6+)

---

## ✨ Features

- 👥 Friend list from JSON data
- 👤 Detailed friend profile page
- 📞 Call, Text, Video interaction system
- ⚡ Timeline tracking of all interactions
- 📊 Friendship analytics using pie chart
- 🔍 Timeline filtering (Call / Text / Video)
- 🔔 Toast notifications on actions
- 📱 Fully responsive design
- ⚠️ 404 page for invalid routes
- ⏳ Loading state while fetching data
